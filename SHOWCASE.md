# LAW TRACK 🏛️

> 대한민국 주요 로펌의 변호사 인력 이동을 실시간으로 추적하는 Intelligence Platform

**작성자**: 황인준
**작성일**: 2026.01.13
**카테고리**: 데이터 분석 & 비즈니스 인텔리전스
**태그**: `Claude Code` `Next.js` `Supabase` `Web Scraping` `Data Visualization`

---

## 💡 왜 LAW TRACK이 필요한가?

### 법률 시장의 숨겨진 문제

매년 수백 명의 변호사가 로펌 간 이동을 합니다. 하지만 이 정보는 파편화되어 있고, 체계적으로 추적할 방법이 없었습니다.

- **로펌 HR 담당자**: "경쟁사가 어떤 인재를 영입했는지, 우리 로펌의 이탈률이 높은지 알 수 없다"
- **헤드헌터**: "어떤 로펌이 공격적으로 확장 중인지, 시장 타이밍을 놓친다"
- **법률 시장 애널리스트**: "수작업으로 데이터를 모으다 보면 이미 오래된 정보"

**LAW TRACK은 이 모든 문제를 해결합니다.**

---

## 🎯 LAW TRACK이 제공하는 가치

### 1️⃣ 매일 자동 업데이트되는 시장 인텔리전스

더 이상 뉴스를 뒤지거나 소문에 의존할 필요가 없습니다.
- ✅ 13개 주요 로펌의 **실시간 인력 변동** 자동 추적
- ✅ 신규 영입, 퇴사, 로펌 간 이직을 **하루 단위로 감지**
- ✅ 매일 아침 최신 데이터로 업데이트 (자동화)

**💼 비즈니스 임팩트**:
*"경쟁사가 시니어 파트너를 영입한 사실을 당일에 파악하여, 우리도 즉시 대응 전략을 수립했습니다."*

---

### 2️⃣ 한눈에 보는 로펌 비교 대시보드

13개 주요 로펌을 카드 한 장씩으로 비교하세요.
- 📊 **현재 변호사 수** + **30일 순변동** 한눈에 비교
- 🔥 **순변동 TOP 5**: 어떤 로펌이 가장 빠르게 성장 중인가?
- 📈 **30일 추세 차트**: 로펌별 인력 변화를 그래프로 시각화

**💡 활용 예시**:
- "김앤장이 이번 달에만 +12명 순증가? 무슨 일이 있었지?"
- "광장과 세종의 30일 트렌드를 비교해서 채용 전략 차이를 파악하자"

---

### 3️⃣ AI가 찾아주는 숨은 패턴

Google Gemini AI가 데이터를 분석해서 인사이트를 제공합니다.
- 🤖 **계절성 패턴**: "매년 1월과 7월에 이직이 급증합니다"
- 🔍 **이동 패턴 분석**: "특정 로펌에서 특정 로펌으로 이동이 많습니다"
- 📌 **이상 징후 탐지**: "이번 달 퇴사율이 평소보다 3배 높습니다"

**🎯 의사결정 지원**:
*"AI 인사이트로 시장 사이클을 파악하고, 우리 로펌의 채용 타이밍을 최적화했습니다."*

---

### 4️⃣ 변호사별 이력 추적

특정 변호사의 커리어 패스를 즉시 조회하세요.
- 👤 **현재 소속 로펌** 및 **직위** 확인
- 📅 **과거 이직 이력** 타임라인
- 🔎 **이름 검색**으로 즉시 찾기

**🔍 헤드헌팅 활용**:
*"관심 있는 후보자의 커리어 패스를 한 번에 파악하고, 접근 전략을 수립합니다."*

## ⚡ 왜 LAW TRACK인가?

### 기존 방식 vs LAW TRACK

| 항목 | 기존 방식 (수작업) | LAW TRACK |
|------|------------------|-----------|
| **데이터 수집** | 뉴스/소문 의존 | 자동 수집 (매일) |
| **커버리지** | 파편화된 정보 | 13개 로펌 전체 |
| **신속성** | 수일~수주 지연 | 당일 감지 |
| **분석** | 수작업 엑셀 | AI 자동 분석 |
| **비용** | 인력 투입 필요 | 구독료만으로 해결 |
| **정확도** | 주관적 판단 | 데이터 기반 객관성 |

**💰 ROI 계산**:
한 명의 애널리스트가 주 10시간을 시장 모니터링에 쓴다면?
- **연간 인건비**: 약 5,000만원 (시간당 10만원 x 10시간 x 50주)
- **LAW TRACK 구독료**: 월 50만원 = 연 600만원
- **절감 효과**: 연 4,400만원 + 더 빠르고 정확한 인사이트

## 🎯 기술 스택 (간단히)

- **프론트엔드**: Next.js 14 + React 19 + TypeScript
- **데이터베이스**: Supabase (PostgreSQL)
- **자동화**: 2단계 Smart Scraping + Vercel Cron
- **AI 분석**: Google Gemini API
- **배포**: Vercel Cloud Platform

**핵심 기술 차별점**:
- ⚡ **2단계 Smart Scraping**: 변동이 없으면 상세 크롤링 생략 → 90% 리소스 절감
- 🤖 **AI 패턴 분석**: 단순 집계를 넘어 숨은 트렌드까지 발견
- 🔄 **완전 자동화**: 설정 한 번으로 매일 최신 데이터

### Claude Code와 함께한 2주간의 여정

**핵심 성과**:
- 📦 **6개 주요 컴포넌트** 설계 및 구현
- 🔧 **스마트 스크래핑 시스템** 아키텍처 설계
- 🎨 **일관된 디자인 시스템** 구축
- 🐛 **TypeScript 타입 안정성** 확보

**기술적 하이라이트**:
1. **2단계 Smart Scraping**: 변동 감지 → 선택적 크롤링 → 90% 리소스 절감

3. **AI 인사이트**: Google Gemini API 연동으로 패턴 자동 분석

이 프로젝트는 AI와 인간이 협력하여 복잡한 비즈니스 문제를 빠르게 해결한 사례입니다.
